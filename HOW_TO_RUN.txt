FitCarousell - Setup & Run Guide
=================================

Prerequisites:
- Node.js 18+
- A Supabase account (free tier works)

1. SUPABASE SETUP
-----------------
a) Create a new project at https://supabase.com
b) Go to SQL Editor and run the contents of supabase/schema.sql
c) Copy your project URL and anon key from Settings > API

2. ENVIRONMENT VARIABLES
------------------------
Create a .env file in the project root:

  VITE_SUPABASE_URL=https://your-project.supabase.co
  VITE_SUPABASE_ANON_KEY=your-anon-key

3. INSTALL & RUN
----------------
  npm install
  npm run dev

The app will start at http://localhost:5173

4. USER ROLES
-------------
- Register as "Individual" to track activities and use the social feed
- Register as "Merchant" to create events and list products for sale

5. KEY FEATURES
---------------
- Real-time activity tracker (Run, Cycle, Swim, Walk) with start/pause/stop
- Manual activity logging
- Community feed with likes, comments, and follows
- Merchant-driven events and marketplace
- Dark-themed Strava-inspired UI
- Responsive design (mobile bottom tabs, desktop top nav)

6. BUILD FOR PRODUCTION
-----------------------
  npm run build
  npm run preview
